# ufibra-crawlers

### Splash
Scrapy-Splash use Splash HTTP API. Check Splash [docs](http://splash.readthedocs.io/en/latest/install.html) for more info.

#### How to run Splash instance

```
$ docker run -p 8050:8050 scrapinghub/splash --max-timeout 3600
```
